# WEB 3.0 (넥스트 이코노미)

분류: IT
저자: 김미경 팀
옮긴이: 김미경 팀
별점: ★★★★☆
: No
한줄평: Web 3.0 완전히 달라진 경제 생태계를 간접경험하고 우리가 왜 Web 3.0 시대를 공부해야하는지 알려주는 책
완독일: 2023년 4월 28일
순서: 64

1. web 1.0
1990년대 등장
read only 개념으로 주로 검색과 문서 + 인터넷의 결합으로 인식한다.모자익으로 시작해서 www, yahoo, google과 같은 검색 엔진등으로 발전
2. web 2.0
2000년대 후반 등장
read write 의 개념으로 사용자가 참여한다는 개념이 크다. 유튜브, 페이스북, 인스타스램,
3. web 3.0
아직 모호한 개념이지만 미래의 웹기술로 인식됨
블록체인, NFT, AI와 같은 신기술에 의한 새로운 개념
디지털 트렌스포메이션이라는 공통점으로 묶고 있지만 아직까지 3.0은 모두가 인정하는 개념은 아님
핵신은 데이터다.
4. 모두가 금융기관이 되는 시대가 온다
개인의 금융자산을 온라인으로 거래할수 있다. 금융기관이 필요가 없어졌다.
5. 스마트폰 앱의 제한
클라우드가 등장하면서 스마트폰 OS의 한계를 극복해 앱으로 하지못하는 웹을 구상하고 있다.
6. 시멘틱 웹
의미론적이란 뜻이면서 동시에 인공지능이 알기 쉽게 만들어진 웹으로 사람과 인공지능이 함께 소셜활동을 하는것.
7. 블록체인과 NFT가 등장한 이유
데이터의 소유권 문제
데이터의 자산권 문제
데어터의 보안문제
이 세가지가 중요해졌고 그것을 해결하기 위해 등장한것이 블록체인과 NFT이다.
8. XR 가상현실이 뒷받침되다.
웹 3.0의 뒷받침 해줄 기술은 XR 메타버스다. 현실보다 훨씬더 안전하고 실용성 높은 메타버스가 대세가 될것.
9. 개인의 시대
개인이 삶에 주체는 내가 된다. 비즈니스의 주체가 플랫폼 기업에서 컨텐츠의 주인으로 점점 이동된다. 개인레게 책임과 함께 이익의 비율이 상승할것.
10. 모두가 공부해야하는 세상
웹 3.0 생태계는 모두가 시작 단계. 그러나 공부하지 않으며 디지털 문맹이 되는 시대.

---

<느낀점>
Web3.0은 아직까진 생소한 느낌이지만 곧 미래의 모습이 될거라 확신이 든다. 미래를 준비하기 위해선 Web3.0의 기술들에 대한 이해도가 필요하고 그에 맞는 공부가 필수적이다. NFT나 블록체인을 비롯해 AI등과 같은 혁신적인 기술들이 대체 되면서 기존에 플랫폼과 중앙시스템에서 벗어나 개인 자체가 산업의 전반을 차지하는 시대가 올것이라 생각되고 그에 맞는 기회가 제공되리라 생각한다.
