# Raw: AI 에이전트 학습 데이터 수집부터 전처리까지 완벽 가이드 (ALCHERA)

헤딩 구조:
1. AI 에이전트 학습 데이터의 정의와 중요성
2. 학습 데이터의 두 가지 유형 (구조화 / 비구조화)
3. 데이터 수집의 3가지 방법 (실제/피드백/합성)
4. 전처리의 중요성과 핵심 단계
5. 학습 데이터 관리 전략
6. 성공과 실패 사례 (금융/의료)

본문 요약:
- 구조화 데이터: DB 정리된 숫자/날짜/문자열, 규칙 기반 솔루션 기초
- 비구조화 데이터: 텍스트/이미지/비디오, 자연어 처리 능력 원천
- 수집 3가지: 실제 환경 데이터 / 인간 피드백 / 합성 데이터
- 전처리: 오류 제거, 비식별화, 일관성 확보
- 관리: 지속 업데이트 + 인간 피드백 통합 + 도메인 특화
- 금융: 거래 패턴 분석 → 사기 예방
- 의료: 의료 데이터 패턴 인식
