## 개요

Hermes Agent는 NousResearch가 만든 오픈소스 AI 에이전트입니다. 2026년 2월 26일에 공개되었고, MIT 라이선스로 누구나 자유롭게 사용할 수 있습니다.

주요 특징:
- 약 60개 내장 도구 포함
- 약 90개 내장 스킬 제공
- 백그라운드에서 대화 리뷰 및 자동 스킬 생성
- 백그라운드 큐레이터(7일 주기 라이브러리 정리)

---

## 활용 사례

1. **Mac Mini 24시간 AI 비서**: Apple 미리알림·메모·iMessage 연동, Telegram 지시, 크론 자동화
2. **팀 디스코드 봇**: 반복 질문 처리, 스킬 축적으로 성능 향상
3. **매일 뉴스 브리핑**: MCP 검색 API 연결, 자동화된 뉴스레터
4. **$5 VPS 무료 운영**: 로컬 LLM(Hermes 3, 8B) 활용 가능

## 핵심 기능

### 대화형 CLI
터미널 `hermes` 명령으로 웹 검색, 파일 작업, 명령 실행 가능

### 유연한 모델 연동
OpenAI, Anthropic, Ollama(로컬 모델), OpenRouter(200개 이상 모델) 등 OpenAI 호환 API면 무엇이든 됩니다.

### 멀티채널 메시징
19개 플랫폼 지원: Telegram, Discord, Slack, WeChat, Teams 등

### 예약 실행과 자동화
자연어 크론 스케줄링 (예: "매일 아침 8시에 뉴스 요약")

### 메모리와 스킬 학습
- 대화 내용 기억
- 작업 절차 스킬로 저장
- 백그라운드 큐레이터 정리

### 영속 목표(`/goal`)
며칠에 걸친 누적 작업 진행, Ralph loop로 백그라운드 점검

### 내장 도구 및 확장
- 약 60개 내장 도구
- Nous Tool Gateway로 웹 검색·이미지 생성·TTS 통합
- MCP로 외부 도구 추가

### Fast Mode와 TUI
- `/fast` 토글로 응답 지연 감소
- React/Ink 기반 새 TUI (`hermes --tui`)
- 로컬 웹 대시보드 (`hermes dashboard`)
