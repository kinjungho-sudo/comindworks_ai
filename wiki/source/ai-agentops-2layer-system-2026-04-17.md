## 원본 자막 요약

비개발자 AI 강사/유튜버가 1개월 동안 업무를 AI 에이전트로 전환한 실전 경험 공유.

**시스템 구성**: 2층 구조
- 업무 비서: Claude Code (정밀한 하네스 엔지니어링)
- 일상 비서: OpenClo (느슨한 하네스, Computer Use 유연)
- 맥미니 24시간 상시 가동, 텔레그램으로 원격 지시

**Claude Code 에이전트 목록 (18개 프로젝트)**:
- 자비스(라우터): 전체 에이전트 분배 허브
- 교육 설계 그룹: 고객사명+직무 입력 → 10가지 실습 시나리오 자동 생성
- 강의 평가 그룹: 수강생 결과물 평가, 개인별 피드백, 결과 보고서
- 강의 슬라이드 그룹: HTML→PPT 변환, 구글 드라이브 업로드 (10분 소요)
- 블로그/유튜브 콘텐츠 제작 (영상 편집, 무음/불필요한 말 정리)
- 링크드인 15인 모니터링 → 매일 정리 브리핑
- 경쟁 유튜브 채널 모니터링 → 주제/썸네일 아이디어 추천
- 이메일 분류 에이전트 → 노션 DB 업데이트, 영업 건 리마인드
- 세금계산서 자동 발행
- 바이브 코딩 (저만의 학습 사이트 ReadWise 벤치마킹)
- 크롤링 에이전트 (쿠팡 리뷰, 나라장터 공고 — 옥토파스보다 뛰어남, 무료)

**OpenClo 일상 비서 활용**:
- 캘린더 관리: 텔레그램으로 지시 → 구글 캘린더 등록, 매일 9시 브리핑
- 카카오톡 나에게 보내기 아카이빙: 하루 2회 자동 수집 → 구글 시트 정리
- KTX 좌석 감시: 안드로이드폰 + ADB + scrcpy로 코레일 앱 제어

**핵심 방법론**:
1. 하네스 엔지니어링: CLAUDE.md + 스킬 + 서브에이전트로 모델 규율
2. 오토리서치(Auto Research): 평가 기준 설정 → AI가 혼자 검증-개선 루프 (안드레 카파시 방법론). PPT 에이전트 1회→10회 적용 결과 품질 극적 향상
3. 깃허브 벤치마킹: 깃허브 링크 주면 Claude Code가 그대로 구현
4. 심층 리서치로 관련 깃허브 레포 찾기
5. Claude Code + Codex 교차 비판: 서로 다른 의견으로 내적 완결성 향상

**결론 메시지**:
- 에이전트 시대 업무 핵심이 "실행"에서 "설계"로 바뀜
- 무엇이 가능한지만 알면 AI가 해준다
- 비개발자도 전문 영역 진입 가능
