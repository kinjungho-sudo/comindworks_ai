# Blueprint Worked Example

This file is a worked example for the blueprint skill.

Use it to calibrate:

- how specific the constraints section should be
- how explicit branching should appear in the flow definition
- how concrete each workflow step's success criteria, validation, and failure handling should be
- how to justify single-agent vs multi-agent structure
- when to add a separate key-decisions section beyond the minimum template

This Codex copy is intentionally lightweight. Treat it as a reference note, not a second template.
