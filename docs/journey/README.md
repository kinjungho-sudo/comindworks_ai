# 코마인드웍스 빌딩 저니

> AI 에이전트 시스템을 직접 만들고, 검증하고, 판매하기까지의 실제 여정.
> 매일 한 것 · 배운 것 · 막힌 것 · 인사이트를 기록한다.

---

## 왜 기록하는가

- **브랜딩**: 내가 직접 쓰는 시스템을 공개하며 신뢰를 쌓는다
- **유튜브 소재**: 각 일지가 영상 1편의 재료
- **학습 가속**: 글로 정리하면 다음 날 더 빠르게 이어간다
- **잠재고객 검증**: 어떤 내용에 반응이 오는지 측정한다

---

## 전체 서사

```
Phase 1: 인프라 구축 (현재)
  → AI 에이전트 시스템 설계 + 자동화 파이프라인 세팅

Phase 2: 첫 번째 결과물
  → 유튜브 대본 파이프라인 실제 가동 + 첫 영상 업로드

Phase 3: 사업화
  → 검증된 시스템을 고객에게 판매 (컨설팅/교육/솔루션)
```

## 새 일지 작성 방법

1. `template.md`를 복사해서 `YYYY-MM-DD.md` 파일 생성
2. 내용 채우기
3. 위 인덱스 테이블에 한 줄 추가
4. `git add . && git commit -m "journey: YYYY-MM-DD Day N"`
